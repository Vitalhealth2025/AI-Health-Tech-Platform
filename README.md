# HealthSmartAI

A personalized nutrition and wellness platform built with Next.js.
Live demo: https://ai-health-tech-platform.vercel.app/dashboard

## About
HealthSmartAI is an AI-driven health platform that helps users track 
their nutrition, log meals, and monitor wellness goals through an 
interactive dashboard.

## Pages
- **Login** — Secure user login
- **Sign Up** — New account creation
- **Dashboard** — Health score, calories, AI recommendations & recent meals
- **Log Meals** — Track daily food intake by meal type
- **Profile** — View and edit personal health information
- **Settings** — App preferences, security and support

## Tech Stack
- [Next.js](https://nextjs.org/) — Frontend framework
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [Vercel](https://vercel.com/) — Deployment
- Firebase — Backend & Authentication *(coming soon)*

## Current Status
UI prototype complete and deployed. Backend integration 
with Firebase in progress.

## Getting Started
Clone the repo and run locally:
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
